# Agent Spec: GCP Knowledge Catalog Agent

## Overview
An intelligent agent demonstrating Google Cloud Platform (GCP) Knowledge Catalog (Data Catalog) capabilities. The agent enables users to search, discover, inspect, and manage data assets (datasets, tables, columns, tags) across GCP projects using the Knowledge Catalog API and remote MCP server integrations. It includes a local web UI / chat interface for rapid interaction and is designed for seamless deployment to Vertex AI Agent Runtime and registration with Gemini Enterprise.

## Example Use Cases
1. **Asset Discovery**: "Find all BigQuery tables containing customer PII in project `my-gcp-project`."
2. **Metadata Inspection**: "Show schema, column descriptions, and classifications for dataset `analytics_prod.orders`."
3. **Tag Management**: "Attach the tag `Confidential` with field `data_owner=security-team` to the table `finance.transactions`."
4. **MCP Tool Integration**: Querying catalog metadata using Knowledge Catalog MCP server tools.

## Tools Required
1. **Knowledge Catalog Search Tool**: Search data assets by query, type, project, or tag.
2. **Asset Metadata & Entry Details Tool**: Fetch detailed entry metadata, schemas, and classifications from GCP Data Catalog / Dataplex.
3. **Tag Attachment & Update Tool**: Create, attach, or update tag templates and tag values on target GCP assets.
4. **Knowledge Catalog Remote MCP Client**: Integrates with remote Knowledge Catalog MCP server for standardized MCP tool invocations.

## Constraints & Safety Rules
- Prompt/confirm before applying destructive modifications or tag deletions on production data assets.
- Validate asset identifiers (entry names, resource paths, tag template IDs) before invoking GCP APIs.
- Adhere to GCP IAM and OAuth authorization boundaries for user access control.
- Never hardcode credentials; authenticate using ambient GCP application default credentials (ADC) or OAuth tokens.

## Success Criteria
- Accurately discover datasets, tables, and columns matching search criteria.
- Successfully retrieve and render asset metadata and schema definitions in readable formatting.
- Reliably attach and update tags on target assets with proper error handling.
- Seamlessly interact via the local web UI and prepare for Gemini Enterprise / Agent Runtime deployment.
