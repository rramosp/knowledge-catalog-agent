The goal of this project is to generate an agent that demonstrates the capabilities of GCP Knowledge Catalog through using its API and remote MCP server. The generated project should contain a local web interface that allows users to interact with the Knowledge Catalog and its capabilities. 

This agent will be deployed in the Gemini Enterprise App and so that users can select it and interact with it using the Gemini Chat UI.

## Key Capabilities to Demonstrate:

* Search and discovery of datasets, tables, columns, and tags across GCP projects.
* Attach and update tags on data assets.
* View metadata, descriptions, and classifications.
* Use the Knowledge Catalog MCP server to search and discover datasets, tables, columns, and tags across GCP projects.
* Handle authentication and authorization seamlessly within the Gemini environment.
